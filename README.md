## A complete, from scratch redesign of a Printables Saturn 4/4 Ultra build plate tilt mount
Features
1. Fixes all geometry issues
2. Addresses a minor build plate collision with the motion column when tilted
3. Improves print-ability
4. Improves part strength

### Instructions
1. Print settings (PETG recommended)
   - .25mm layers
   - 4 top and bottom layers
   - 5 walls
   - Support threshold angle of 38 degrees
   - As little as 10% infill
2. Two M4 x 18mm stainless flat head wood screws required
   - Replaces the 15mm center screws on the top of the build plate
3. Ensure your filament profile is dialed in
   - If the shrinkage modifier is off by more than .25% the fit will not be perfect as designed

### Installation
1. Remove the plastic handle wings on the top of the build plate
2. Remove the center screw on the top edge of the front and back of the build plate
3. Place the printed shim, chamfer down, into the rear center screw pocket
4. Install the Saturn 4 Ultra Build Plate Tilt Dripper over the top of the build plate
   - Orient so the angled receptacle is to the right of the build plate 
5. Install the two M4 x 18mm screws to secure the part
   
Original model can be found [here](https://www.printables.com/model/925187-saturn-4-ultra)
